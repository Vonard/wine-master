# Новое русское вино

Сайт магазина авторского вина "Новое русское вино".

### Требования

Для запуска сайта вам понадобится Python версии 3.10 или выше.

Проверьте, установлен ли у вас Python, выполнив команду в терминале: `python --version` (или `python3 --version`).
Если Python не установлен, скачайте его с [официального сайта](https://www.python.org/downloads/) и установите. 
При установке на Windows обязательно поставьте галочку "Add Python to PATH".

### Как установить

Клонируйте репозиторий на свой компьютер:
   
```
git clone https://github.com/Vonard/wine-master
cd wine-master
```
 
Затем используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей:
   
```
pip install -r requirements.txt
```  

### Переменные окружения

Для работы сайта нужна таблица продукции. Создайте в папке проекта файл `.env` и вставьте следующее:

```
WINE_TABLE=путь_до_файла_таблицы.xlsx
```

### Как запустить

Чтобы запустить сайт, в командной строке перейдите в папку проекта и выполните команду:

```
python main.py
```

Перейдите на сайт по адресу [http://127.0.0.1:8000](http://127.0.0.1:8000).
